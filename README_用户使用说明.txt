SenseVoice 语音识别系统 - 用户使用说明
==========================================

欢迎使用 SenseVoice 语音识别系统！

📋 系统要求
-----------
- Windows 10/11 或 macOS 10.14+ 或 Linux
- 至少 4GB 内存（推荐 8GB+）
- 约 2GB 磁盘空间（用于模型文件）

🚀 首次使用
-----------
1. 双击运行 SenseVoice.exe (Windows) 或 SenseVoice.app (macOS)

2. 首次运行会自动下载模型文件（约 900MB）
   - 需要网络连接
   - 下载时间取决于网速（通常 5-15 分钟）
   - 下载完成后可离线使用

3. 模型下载完成后，会自动打开浏览器
   - 如果没有自动打开，请访问: http://localhost:7860

💡 使用方法
-----------
1. 实时录音识别
   - 点击"实时录音识别"标签页
   - 点击录音按钮开始说话
   - 录音结束后会自动识别（或点击"开始识别"按钮）
   - 识别结果会显示在右侧

2. 上传音频文件
   - 点击"上传音频文件"标签页
   - 点击上传区域选择音频文件
   - 支持格式: wav, mp3, flac 等
   - 点击"开始识别"按钮
   - 识别结果会显示在右侧

⚠️ 注意事项
-----------
- 首次运行需要网络连接下载模型
- 模型下载后可以完全离线使用
- 关闭程序时，Web 界面也会关闭
- 如果遇到问题，请检查防火墙设置

🛠️ 常见问题
-----------
Q: 程序无法启动？
A: 请检查系统是否满足要求，尝试以管理员权限运行

Q: 模型下载失败？
A: 请检查网络连接，或使用 VPN/代理

Q: 浏览器无法打开？
A: 请手动访问 http://localhost:7860

Q: 识别不准确？
A: 确保录音清晰，环境安静，说话清晰
